# Future Features

> Ideas and plans for post-MVP development.

---

## Near-term (Next Release)

| Feature | Priority | Notes |
|---------|----------|-------|
| Archive/Trash | High | Soft-delete for references and projects |
| Documents (PDF, images) | High | Owned file storage with preview |
| Provider-specific metadata | Medium | GitHub stars, YouTube duration, etc. |
| Mobile Share Sheet | High | One-tap capture from any app |
| Keyboard shortcuts | Medium | Sidebar navigation, batch operations |

## Medium-term

| Feature | Priority | Notes |
|---------|----------|-------|
| Full-text search | High | Search within saved page content |
| Related knowledge suggestions | Medium | Based on shared tags and content similarity |
| Duplicate detection | Medium | Avoid saving the same URL twice |
| Bulk operations | Medium | Multi-select, batch move/tag/delete |
| Project templates | Low | Pre-built project structures |
| Export/import | Medium | Full vault export as JSON |

## Long-term

| Feature | Priority | Notes |
|---------|----------|-------|
| Multi-user / teams | Low | Shared workspaces, collaborative tags |
| Public API | Medium | Third-party integrations |
| Plugin system | Low | Extend functionality via plugins |
| Offline mode | Medium | Local-first sync |
| Knowledge graph | Low | Visual exploration of connections |
| AI recommendations | Low | Personalized discovery based on usage |

## Deliberately NOT Planned

- Real-time collaboration (Google Docs-style)
- AI chat assistant (use ChatGPT/Claude directly)
- Email client
- Note-taking app features (type /template, etc.)
- Project management features (sprints, assignments, deadlines)
