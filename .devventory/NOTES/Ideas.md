# Ideas

> Random product ideas and observations. Not commitments.

---

## Capture

- **Read-later workflow**: Mark references as "to read" → "reading" → "read". Track reading time.
- **Email capture**: Forward emails to a Devventory inbox → create reference.
- **Twitter/X capture**: Save tweets and threads directly.
- **Highlight from page**: Select text on any page → save as quote with URL.

## Organization

- **Smart folders**: Auto-folder by domain, category, or tag pattern.
- **Nested tags**: Tag hierarchies (e.g., `frontend/react`, `backend/node`).
- **Saved searches**: Named search queries that appear in sidebar.
- **Auto-tagging rules**: "When saving from github.com, add 'github' and 'code' tags."

## Retrieval

- **Full-text search**: Search within saved page content, not just title/metadata.
- **Semantic search**: Embedding-based similarity search.
- **Related content**: "People who saved this also saved..." recommendations.
- **Weekly digest**: Email summary of new saves, stale items, trending tags.

## UI

- **Quick capture from anywhere**: Global keyboard shortcut (even when app not focused).
- **Split view**: Side-by-side reference comparison.
- **Mind map**: Visual tag/topic browser.
- **Command palette extensions**: Plugins can add commands to Cmd+K.

## Integration

- **Zapier / n8n**: Connect Devventory to any tool.
- **Obsidian sync**: Bidirectional sync with Obsidian vault.
- **Readwise import**: Import existing highlights and annotations.
- **API for developers**: Build custom tools on top of Devventory.

## Monetization (Future)

- **Cloud sync**: Premium tier for instant sync across devices.
- **Team workspaces**: Shared folders, references, and projects.
- **Custom AI model**: Train on user's knowledge base for personalized suggestions.
- **API rate limits**: Free tier has limits, pro tier has unlimited access.
