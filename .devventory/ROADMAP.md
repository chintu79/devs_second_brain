# Roadmap

> **Current phase:** MVP — validating that people want one place to capture, organize, and retrieve everything important.

---

## Current MVP

### Done
- [x] Global architecture restructure (11 → 4 pages)
- [x] Knowledge as References workspace (three-panel layout, folder tree, inline editor, backlinks)
- [x] Dashboard as continuation page (Continue Working + Recent Captures)
- [x] Projects as reference-only workspaces (no kanban, no prompts/notes)
- [x] Browser extension (WXT framework, Chrome MV3, capture URLs + context)
- [x] AI enrichment (metadata generation, summaries, auto-labels)
- [x] Universal reference model (URL, title, metadata, tags, folders, notes)
- [x] Cmd+K global search
- [x] Always-editable content (lock/unlock removed)
- [x] Landing page simplified (hero + CTA only)

### In Progress
- [ ] Folder tree files inside folders (VS Code/Finder style)
- [ ] Browser extension mobile support
- [ ] Mobile capture via Share Sheet

### Next Up
- [ ] Archive/Trash functionality
- [ ] VSCode-style right-click context menu on files
- [ ] Keyboard navigation for sidebar
- [ ] Back/forward navigation history

---

## Next Release
